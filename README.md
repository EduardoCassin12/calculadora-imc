# Calculadora de IMC

Projeto desenvolvido em JavaScript para calcular o Índice de Massa Corporal (IMC) utilizando peso e altura.

## 📋 Objetivo

Calcular o IMC utilizando a fórmula:

IMC = Peso / (Altura × Altura)

## 🚀 Tecnologias

- JavaScript
- Node.js

## 📚 Conceitos praticados

- Variáveis
- Operadores matemáticos
- Estruturas condicionais
- `console.log()`

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/EduardoCassin12/calculadora-imc.git
```

2. Entre na pasta:

```bash
cd calculadora-imc
```

3. Execute:

```bash
node imc.js
```
