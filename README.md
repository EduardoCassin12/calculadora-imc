# Calculadora de Desconto

Projeto desenvolvido em JavaScript para calcular o valor do desconto aplicado a um produto, exibir o preço final e informar a classificação da promoção.

## 📋 Objetivo

Calcular o valor do desconto de um produto com base em uma porcentagem informada, exibir o preço final e classificar a promoção conforme o percentual de desconto.

## 🚀 Tecnologias

- JavaScript
- Node.js

## 📚 Conceitos praticados

- Variáveis
- Operadores matemáticos
- Estruturas condicionais (`if/else`)
- Template Strings
- Formatação de números com `toFixed()`

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/EduardoCassin12/calculadora-descontos.git
```

2. Entre na pasta do projeto:

```bash
cd calculadora-desconto
```

3. Execute o arquivo principal:

```bash
node src/index.js
```

## 📷 Exemplo de saída

```text
O preço era de: R$ 250.00
```
